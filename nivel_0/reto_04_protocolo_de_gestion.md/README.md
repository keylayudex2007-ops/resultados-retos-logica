# Protocolo de Gestión de Compuertas Industriales

## Validación de Emergencia (Nivel Crítico)
**¿Nivel > 120m?**
* **SÍ:** Abrir Compuerta A + B y Alarma General.
* **NO:** Evaluar Nivel Preventivo.

---

## Validación Preventiva
**¿Nivel > 100m?**
* **SÍ:** Evaluar variable de **Velocidad**.
    * **Alta:** Abrir Compuerta A.
    * **Media:** Abrir Compuerta B.
    * **Baja:** Compuertas Cerradas.
* **NO:** Compuertas Cerradas.

---
**Resultado:** Fin del proceso.
